# equals.com — Design Reference

## 1. Visual Theme & Atmosphere

Equals occupies the intersection of data utility and refined brand identity. The design communicates trustworthiness through a warm off-white canvas (`#FAF9F5`) rather than cold pure white — a subtle but meaningful choice that evokes paper and craft. The typographic system pairs a condensed serif (**Serrif Condensed**) for display with **Unica77** (a Swiss grotesque) for body, creating tension between editorial warmth and functional clarity.

The color system is rich and named — Orchid, Glacier, Peach, Amber — suggesting a brand that took care to name its palette. This is a product that knows who it is.

## 7. Do's and Don'ts

**Do:**
- Use the warm off-white (`#FAF9F5`) as the canvas — never pure white
- Pair Serrif Condensed display with Unica77 body copy
- Use pill-shaped buttons (60px border-radius) for primary actions
- Reserve Orchid and Amber as accents — use sparingly
- Use Aeonik Fono uppercase for small labels and section markers

**Don't:**
- Add decorative shadows — this is a flat, layered system
- Use more than one accent color per component
- Set body copy in the display serif — it belongs only at headline scale
- Mix the warm background with pure white surfaces in the same view

## 8. Responsive Behavior

- Three primary breakpoints: 700px (tablet), 1100px (narrow desktop), 1280px (full)
- Headlines scale down but Serrif Condensed remains the display face
- Layout narrows to single column at mobile breakpoints
- Pill button shapes are preserved at all sizes
