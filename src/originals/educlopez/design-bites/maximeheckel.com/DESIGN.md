# maximeheckel.com — Design System Reference

## 1. Visual Theme & Atmosphere

Dark scholarly elegance with typographic sophistication. This is a site where typeface choices tell the full story: Instrument Serif for display (warmth, literary authority), iA Writer Quattro V for longform reading (monospace clarity, writer's tool), and Inter for UI. The palette operates in perceptually uniform oklch — a deep near-black background with cool muted blue-gray hues. The inset shadow on interactive elements (white glow top + dark shadow bottom) creates a tactile pressed-glass effect unique to this site.

## 7. Do's and Don'ts

**Do:**
- Use the exact colour roles as defined — each serves a specific semantic purpose
- Respect the typeface role boundaries (each font has one job)
- Preserve generous whitespace — crowding breaks the design language
- Keep interactions restrained: hover opacity or colour-shift, not complex transforms

**Don't:**
- Introduce colours outside the defined palette
- Mix typefaces beyond the established pairing
- Add drop shadows where the design is intentionally flat
- Use gradients unless the site already employs them
- Add Colour accents
- Add light mode
