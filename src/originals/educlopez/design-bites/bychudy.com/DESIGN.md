# DESIGN.md — bychudy.com

## 1. Visual Theme & Atmosphere

Bychudy is an ultra-minimal Polish design portfolio that treats the screen like a blank page. The design philosophy is radical reduction: system fonts, black-and-white, uppercase micro-labels, and almost no decorative surface treatment. The atmosphere is "type-led, brutalist in its honesty." Every design decision here is an act of removal rather than addition. The site communicates confidence through what it refuses to do.

## 3. Typography Rules

**Typeface**: `SF Pro Text Bold` (Apple system font) for h1, `sans-serif` generic fallback for body and links.

| Selector | Size | Weight | Notes |
|----------|------|--------|-------|
| h1 | 13px | 400 | System font at label scale — with `text-transform: uppercase` |
| body | 12px | 400 | Below standard web body size |
| a | 12px | 400 | Matches body exactly |

- **Uppercase h1 at 13px** is the defining typographic gesture: headings are not large — they are labeled.
- The use of SF Pro Text Bold but `font-weight: 400` is unusual — the "Bold" variant is baked into the font name itself.
- 12px body text is a deliberate low-information-density choice — each line carries weight.
- Line height of 15.6px on h1 creates very tight label-style spacing.
- **What's absent**: no custom web fonts, no large display sizes, no tracking adjustments, no letter-spacing artistry.

No depth mechanism detected. This site is perfectly flat. The only "elevation" is the presence/absence of image content beneath text. White translucent overlays (`rgba(255,255,255,0.17)`) suggest text panels placed over photography — transparency creates a sense of float without a shadow.

## 7. Do's and Don'ts

**Do:**
- Use uppercase 13px SF Pro Text for all label-scale headings.
- Keep body at 12px — the density is intentional.
- Leave links unstyled (browser default blue) if pursuing this aesthetic.
- Introduce color (the orange) only once, at maximum emotional impact.

**Don't:**
- Add web fonts — SF Pro Text and system sans-serif are the design.
- Add hover shadows or card depth — flatness is the statement.
- Increase font sizes — the micro-scale is part of the tension.
- Use rounded corners — sharp or default only.

## 8. Responsive Behavior
