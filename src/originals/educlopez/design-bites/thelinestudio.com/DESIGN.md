**Do:** Use DenimWeb at extreme scale for hero/display. Apply the `"ss03"` font feature universally. Let the cool gray canvas breathe — don't fill it with color. Use red only for the brand moments (active nav, footer).

**Don't:** Add intermediate heading sizes — the macro/micro contrast is the system. Don't add border radius — sharp edges are essential. Don't soften the red with transparency.
