# wabi.ai — Design System

## 1. Visual Theme & Atmosphere

Minimal AI product landing with high-contrast dark mode. The design communicates speed and intelligence through stark typographic contrast — a custom display font with extreme negative tracking on headlines against a near-black canvas.

**Mood:** dark, bold, confident, tech-forward

The visual language is refined without being austere — custom typefaces carry the personality while structural clarity does the heavy lifting. Colors are applied directly without a formal token layer — values are contextual rather than systematic.

---

## 2. Color Palette & Roles

**Palette philosophy:** Dark-first — the background is the canvas; light only appears for contrast hierarchy.

## 5. Layout Principles





**Layout philosophy:** Content-first layout — sections breathe independently; no fixed grid forces awkward compositions.

Spacing cadence: Direct pixel values — consistent multiples of 4px or 8px observed in component padding.

---

## 6. Depth & Elevation

**Flat design** — no box shadows. Depth communicates through:
- Background color contrast between sections
- Border lines as separators
- Typography weight shifts
- Proximity and whitespace

This is an intentional choice: shadows would add visual noise to a carefully restrained palette.

---

## 7. Do's and Don'ts

**Do:**
- Maintain the dark background as the dominant canvas — light is a highlight, not the base
- Use only the defined typefaces (selecta) — system fonts will break the brand voice
- Keep all edges sharp — border-radius destroys the visual discipline
- Keep elevation flat — adding shadows would contradict the design language

**Don't:**
- Add light mode without a full color system rethink — the palette is built for dark
- Use accent colors for decoration — they are reserved for interactive signals only
- Mix more than two typeface families — typographic chaos undermines brand clarity
- Add gradient backgrounds unless they exist in the design system
- Use stock UI component defaults (e.g., Bootstrap/MUI) without full restyling
