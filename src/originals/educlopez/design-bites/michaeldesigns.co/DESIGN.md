# michaeldesigns.co — Design System

## 1. Visual Theme & Atmosphere

Michael Designs presents as a no-nonsense portfolio with a strong editorial voice. The palette is deliberately tight — near-black text on white, punctuated by vivid accent shots that announce personality without undermining credibility. The design feels like a printed publication translated to the web: generous margins, disciplined type hierarchy, and a coral-red that jolts the eye exactly when intended. There is no decoration for its own sake. Every element earns its place.

## 3. Typography Rules

**Typeface:** Market Sans (Helvetica fallback) — a custom grotesque that lends an agency-grade confidence throughout.

| Context | Size | Weight | Notes |
|---|---|---|---|
| Body / base | 16px | 400 | Comfortable reading measure |
| H2 headings | 21px / 31.5px lh | 700 | Bold, tight hierarchy |
| H3 subheadings | 14px | 700 | Small-caps-style weight bump |
| UI / select | 12px | 400 | 16px line height, compact |

- No letter-spacing customisation — the typeface ships with balanced defaults.
- No `text-transform` — the designer trusts the letterforms.
- Heading weight jumps (400→700) do the heavy lifting rather than size alone.

- **Buttons:** No shadow. Flat, colour-block style. Colour conveys state, not elevation.

## 6. Depth & Elevation

Essentially flat. No box shadows detected on primary surfaces. The design uses **colour contrast** and **typographic weight** to create visual hierarchy instead of elevation or blur effects. The coral and red accents provide the only "lift" — achieved through chromatic pop rather than shadow.

## 7. Do's and Don'ts

**Do:**
- Use Market Sans at bold weights for headings — it defines the brand voice.
- Apply the coral accent sparingly for maximum impact.
- Maintain the 48px horizontal padding on the body for editorial breathing room.
- Use flat, solid-fill buttons. No gradients.

**Don't:**
- Add box shadows or card elevations — the design is intentionally flat.
- Introduce new typefaces. The single-font system is part of the identity.
- Use blue or cool colours — the warm palette is coherent and intentional.
- Overcrowd layouts. Whitespace is a first-class design element here.
