# eryc.cc — Design Reference

## 1. Visual Theme & Atmosphere

eryc.cc is a personal portfolio built with quiet confidence. The design philosophy is anti-spectacle: a soft gray canvas (`#F6F7F7`), muted mid-tone text, and a single typeface family (Roobert) used at modest scale across all elements. There is no hero typography, no scroll-jacking, no animation library. The atmosphere is contemplative and honest — a designer who doesn't need to shout.

Built on Astro, the site signals a developer-designer hybrid. The font feature settings (ss02, ss03, ss04, ss07, ss10, ss11) on every element show obsessive attention to Roobert's OpenType possibilities.

## 6. Depth & Elevation

No shadows, no cards, no elevation at all. This is the flattest possible design — a single plane with only tonal variation in text color. Depth is achieved purely through contrast: dark text on light background, muted secondary text on the same ground.

## 7. Do's and Don'ts

**Do:**
- Use Roobert with its full OpenType feature set (ss02–ss11)
- Keep the gray scale tight — 9 steps, cool tints
- Use 15px/500 as the base text size — larger than typical web body copy
- Rely on position and whitespace alone for hierarchy, not type scale
- Use wide horizontal margins (60px+)

**Don't:**
- Add any brand color — this is a grayscale-only system
- Use decorative elements, shadows, or gradients
- Set headlines larger than ~20px — the design is anti-spectacle
- Add hover animations — if interactions exist, they are subtle

## 8. Responsive Behavior
