# SKILL: Checkbox, Radio & Toggle

## Source
`DesignX Catalog`

## Purpose
Дизайн элементов выбора: checkbox, radio button, toggle/switch. Custom controls, indeterminate state, группа, accessibility.

## When to Use
- Проектирование и разработка интерфейсов в категории interaction
- Работа над продуктами: web-app, saas, mobile, desktop, dashboard
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Checkbox: множественный выбор, indeterminate, tri-state
4. - Radio: взаимоисключающий выбор, группа, default selected
5. - Toggle: бинарный выбор, immediate action, label справа/слева
6. - Custom: 44×44px touch target, focus ring, disabled, aria-*

checkbox, radio, toggle, switch, custom-controls, accessibility
