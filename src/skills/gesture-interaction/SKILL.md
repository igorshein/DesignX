# SKILL: Gesture Interaction

## Source
`DesignX`

## Purpose
Жестовое взаимодействие: swipe, pinch, long-press, haptic feedback, multi-touch gestures, discoverability.

## When to Use
- Проектирование и разработка интерфейсов в категории interaction
- Работа над продуктами: mobile, web-app, desktop
- Проектирование тач-интерфейсов

## Behavior
1. Осмысли контекст: платформу, устройство, контекст использования
2. Следуй ключевым принципам:
3. - Жесты должны быть обнаруживаемыми
4. - Визуальные подсказки для скрытых жестов
5. - Haptics подтверждают действия, а не объявляют о них
6. - Поддерживай accessibility: жесты + кнопки
7. - Предотвращай случайные срабатывания порогами

## Tags
gesture, swipe, pinch, haptics, touch

## For Product Types
mobile, web-app, desktop
