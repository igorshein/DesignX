# SKILL: Product Analytics for UX

## Source
`DesignX`

## Purpose
Product-аналитика для UX-решений: Amplitude, Mixpanel, funnels, cohorts, product-led growth метрики.

## When to Use
- Проектирование и разработка интерфейсов в категории ux-research
- Работа над продуктами: web-app, saas, dashboard, mobile, ecommerce
- Data-driven дизайн-решения

## Behavior
1. Осмысли контекст: продуктовые цели, события, воронки, когорты
2. Следуй ключевым принципам:
3. - Отслеживай события, а не просмотры страниц
4. - Воронки показывают точки оттока, а не только конверсию
5. - Когорты выявляют паттерны удержания
6. - Product-led growth требует метрик активации
7. - Всегда связывай аналитику с дизайн-решениями
