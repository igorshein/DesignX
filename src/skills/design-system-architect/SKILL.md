# SKILL: Design System Architect

## Source
`designer-skills` · [GitHub](https://github.com/Owl-Listener/designer-skills)

## Purpose
Целостная дизайн-система: от атомарных токенов до сложных паттернов. Структурирование палитры, шкалы типографики, отступов, иконок, анимаций и документации.

## When to Use
- Проектирование и разработка интерфейсов в категории design-system
- Работа над продуктами: web-app, saas, mobile, ecommerce, desktop
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Атомарный дизайн: tokens → components → patterns → pages
4. - Design tokens: цвета, отступы, типографика как переменные
5. - Документация принципов, voice & tone, бренд-гайдов
6. - Разделение на product UI и brand toolkit
