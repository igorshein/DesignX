# SKILL: Responsive Design Principles

## Source
`DesignX Catalog`

## Purpose
Responsive design: breakpoints (576, 768, 992, 1200), fluid grids (%), flexible images, media queries, mobile-first подход, container queries.

## When to Use
- Проектирование и разработка интерфейсов в категории responsive
- Работа над продуктами: web-app, landing, saas, mobile, ecommerce, dashboard
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Breakpoints: 576px mobile, 768px tablet, 992px desktop, 1200px wide
4. - Fluid: %, rem, vw вместо px
5. - Mobile-first: min-width media queries
6. - Images: max-width: 100%, srcset для density
