# SKILL: Cognitive Mapping

## Source
DesignX Catalog · ux-research

## Purpose
Построение ментальных карт пользователей: когнитивные модели, восприятие информации, card sorting, tree testing.

## When to Use
- Проектирование и разработка интерфейсов в категории ux-research
- Работа над продуктами: web-app, saas, mobile, dashboard, ecommerce
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Ментальные модели: выявление скрытых структур восприятия
4. - Card sorting: открытая и закрытая сортировка, анализ дендрограмм
5. - Tree testing: проверка findability, task-based оценка навигации
6. - Синтез: информационная архитектура, категоризация, метафоры
