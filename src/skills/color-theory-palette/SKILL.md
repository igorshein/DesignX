# SKILL: Color Theory & Palette Generation

## Source
DesignX Catalog · color-theory

## Purpose
Работа с цветовыми моделями (HSL, RGB, LAB), построение гармоничных палитр (комплементарные, аналогичные, триадные), подбор акцентных цветов, генерация шкалы 50→900.

## When to Use
- Проектирование и разработка интерфейсов в категории color-theory
- Работа над продуктами: web-app, dashboard, landing, saas, mobile, ecommerce, brand
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Цветовой круг: комплементарные, аналогичные, триадные схемы
4. - HSL-модель: hue, saturation, lightness для осмысленных палитр
5. - 60-30-10 правило: основной, вторичный, акцентный цвета
6. - Генерация шкалы от primary: 50→900 (светлые→тёмные)
