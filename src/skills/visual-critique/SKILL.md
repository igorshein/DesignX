# SKILL: Visual Critique

## Source
`designer-skills` · [GitHub](https://github.com/Owl-Listener/designer-skills)

## Purpose
Анализ визуальной иерархии, проверка консистентности бренда, оценка композиции и аудит типографики.

## When to Use
Когда нужно провести ревью готового макета или существующего интерфейса, выявить проблемные зоны.

## Behavior
1. Определи, что выделяется в первую очередь — соответствует ли это задачам интерфейса
2. Проверь консистентность бренда по mood/voice/tokens
3. Оцени композицию: баланс, ритм, распределение веса
4. Проверь типографику: размеры, интерлиньяж, контраст
5. Дай структурированный фидбек с указанием конкретных проблемных зон
6. Приоритизируй: критические → важные → косметические
