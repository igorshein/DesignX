# SKILL: In-App Messaging

## Source
`DesignX`

## Purpose
In-app уведомления: тосты, snackbar, banners, modal notifications, notification center, приоритетная система.

## When to Use
- Проектирование и разработка интерфейсов в категории interaction
- Работа над продуктами: web-app, mobile, saas, desktop
- Проектирование системы in-app коммуникации

## Behavior
1. Осмысли контекст: типы сообщений, приоритеты, сценарии использования
2. Следуй ключевым принципам:
3. - Toast для успеха, snackbar для undo
4. - Banners для постоянных сообщений
5. - Notification center для истории
6. - Auto-dismiss с адекватным временем чтения
7. - Стек сообщений, а не наложение
