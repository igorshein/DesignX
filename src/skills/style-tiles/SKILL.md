# SKILL: Style Tiles

## Source
`DesignX Catalog`

## Purpose
Промежуточный артефакт между мудбордом и дизайном: цвета, шрифты, текстуры, стиль кнопок/форм/иконок, mood. 3 варианта для обсуждения.

## When to Use
- Проектирование и разработка интерфейсов в категории visual-design
- Работа над продуктами: web-app, landing, saas, brand, marketing
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Показывает: цвета, шрифты, текстуры, UI-элементы
4. - 3 варианта: 2 контрастных + 1 безопасный
5. - Формат: Figma board или HTML/CSS
6. - Без full layout — только визуальный язык
