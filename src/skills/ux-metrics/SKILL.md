# SKILL: UX Metrics & KPIs

## Source
`DesignX Catalog`

## Purpose
Определение UX-метрик: A.C.T. (Attitude, Cognition, Task), HEART framework (Happiness, Engagement, Adoption, Retention, Task success), baseline + target.

## When to Use
- Проектирование и разработка интерфейсов в категории ux-research
- Работа над продуктами: web-app, saas, mobile, ecommerce, dashboard
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Behavioural: success, time, errors, clicks
4. - Attitudinal: SUS, NPS, CSAT, UEQ
5. - Outcome: retention, conversion, churn, DAU/MAU
6. - HEART: Happiness, Engagement, Adoption, Retention, Task
