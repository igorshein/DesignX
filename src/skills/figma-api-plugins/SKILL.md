# SKILL: Figma API & Plugins Dev

## Source
DesignX Catalog · plugins

## Purpose
Разработка Figma-плагинов и работа с REST API: автоматизация повторяющихся задач, кастомные плагины, интеграция дизайн-токенов в CI/CD.

## When to Use
- Проектирование и разработка интерфейсов в категории design-tools
- Работа над продуктами: web-app, saas, desktop
- Интеграция с дизайн-системами и CI/CD пайплайнами

## Behavior
1. Осмысли контекст: Figma-файл, API-эндпоинты, целевая автоматизация
2. Следуй ключевым принципам:
3. - Figma Plugin API для интерактивных плагинов
4. - REST API для CI/CD интеграции
5. - Widgets для real-time коллаборации
6. - Экспорт токенов в код (style dictionary, design tokens)
7. - Component creation и auto-layout генерация
