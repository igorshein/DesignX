# SKILL: Theme Factory

## Source
`anthropics/skills` · [GitHub](https://github.com/anthropics/skills)

## Purpose
Стилизация артефактов по профессиональным темам или генерация кастомных тем.

## When to Use
Когда нужно быстро изменить визуальный стиль артефакта под конкретный бренд или создать новую тему.

## Behavior
1. Проверь наличие DESIGN.md или brand-описания
2. Если есть — примени готовые токены из DESIGN.md
3. Если нет — выбери из библиотеки тем (Linear, Stripe, Vercel, Apple и др.)
4. Сгенерируй CSS-переменные под тему
5. Экспортируй DESIGN.md с токенами созданной темы
