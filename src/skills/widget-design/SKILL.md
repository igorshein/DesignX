# SKILL: Widget Design

## Source
`DesignX Catalog`

## Purpose
Дизайн виджетов: home screen widgets, lock screen widgets, glanceable UI, widget families, iOS/Android conventions.

## When to Use
- Проектирование и разработка интерфейсов в категории mobile
- Работа над продуктами: mobile, web-app, desktop
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Glanceable UI: информация за 3 секунды, минимализм
4. - Размеры: small/medium/large/lock screen, widget families
5. - iOS WidgetKit: timeline provider, configurable intent, SwiftUI
6. - Android: AppWidgetProvider, responsive layout, configuration activity
