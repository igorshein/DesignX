# SKILL: Visual Balance & Symmetry

## Source
DesignX Catalog · layout-composition

## Purpose
Создание сбалансированных композиций: симметрия (стабильная), асимметрия (динамичная), оптическая компенсация веса элементов.

## When to Use
- Проектирование и разработка интерфейсов в категории layout-composition
- Работа над продуктами: web-app, dashboard, landing, saas, mobile, ecommerce, desktop, brand
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Симметрия = формальный, надёжный, предсказуемый
4. - Асимметрия = динамичный, современный, вовлекающий
5. - Оптический вес: тёмные цвета тяжелее светлых
6. - Компенсация: маленький элемент уравновешивает большой цветом
