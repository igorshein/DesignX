# SKILL: Microcopy Design

## Source
`DesignX`

## Purpose
Дизайн UI-текстов: кнопки, error messages, placeholder, tooltips, empty states, confirmation dialogs.

## When to Use
- Проектирование и разработка интерфейсов в категории brand
- Работа над продуктами: web-app, mobile, saas, ecommerce, landing, desktop
- Написание интерфейсных текстов

## Behavior
1. Осмысли контекст: пользовательский сценарий, эмоциональное состояние, цель
2. Следуй ключевым принципам:
3. - Каждое слово имеет задачу — остальное вырезать
4. - Error messages предлагают решение, а не извиняются
5. - Кнопки говорят, что произойдёт, а не «нажми здесь»
6. - Empty states направляют к следующему действию
7. - Placeholder показывает пример, а не инструкцию
