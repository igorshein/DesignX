# SKILL: Expert Review

## Source
DesignX Catalog · ux-research

## Purpose
Экспертный обзор интерфейса силами domain expert: эвристическая оценка, когнитивный walkthrough, предметная экспертиза.

## When to Use
- Проектирование и разработка интерфейсов в категории ux-research
- Работа над продуктами: web-app, dashboard, saas, desktop, mobile
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Эвристическая оценка: 10 эвристик Nielsen, severity rating
4. - Когнитивный walkthrough: задачи пользователя, шаги, обратная связь
5. - Domain expert analysis: бизнес-логика, отраслевые стандарты
6. - Отчёт: проблемы с приоритетами, рекомендации, референсы
