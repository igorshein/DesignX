# SKILL: Dark/Light Mode Design

## Source
DesignX Catalog · visual-design

## Purpose
Проектирование тёмных и светлых тем: WCAG-контраст, палитра для тёмного фона, elevation, prefers-color-scheme переключение.

## When to Use
- Проектирование и разработка интерфейсов в категории visual-design
- Работа над продуктами: web-app, dashboard, saas, mobile, ecommerce, desktop
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Насыщенные цвета в тёмных темах часто не проходят WCAG 4.5:1
4. - Более светлые оттенки для текста на тёмном фоне
5. - Elevation через цвет фона, а не тени
6. - prefers-color-scheme для системного переключения
