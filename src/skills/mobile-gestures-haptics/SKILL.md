# SKILL: Mobile Gestures & Haptics

## Source
`DesignX Catalog`

## Purpose
Тактильная обратная связь и системные жесты в мобильных интерфейсах: haptic feedback, gesture recognition, iOS/Android patterns.

## When to Use
- Проектирование и разработка интерфейсов в категории mobile
- Работа над продуктами: mobile, web-app, desktop
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Системные жесты: swipe, pinch, long-press, double-tap, drag
4. - Haptic feedback: light/medium/heavy impact, notification, selection
5. - iOS UIFeedbackGenerator vs Android HapticFeedbackConstants
6. - Edge gestures: system back, home indicator, reachability
