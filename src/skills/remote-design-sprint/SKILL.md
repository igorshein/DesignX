# SKILL: Remote Design Sprint

## Source
`DesignX`

## Purpose
Удалённые дизайн-спринты: FigJam facilitation, async collaboration, sprint planning, remote workshop техники.

## When to Use
- Проектирование и разработка интерфейсов в категории design-ops
- Работа над продуктами: web-app, saas, mobile, desktop, landing
- Проведение удалённых воркшопов и спринтов

## Behavior
1. Осмысли контекст: распределение команды, часовые пояса, цели спринта
2. Следуй ключевым принципам:
3. - Async-фазы уважают часовые пояса
4. - FigJam-шаблоны ускоряют настройку
5. - Фасилитация важнее в удалённом формате
6. - Документируй каждое решение немедленно
7. - Результаты спринта важнее артефактов
