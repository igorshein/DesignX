# SKILL: Mobile-First Design

## Source
`DesignX Catalog`

## Purpose
Стратегия mobile-first: начинать с самого маленького экрана, прогрессивно улучшать. min-width media queries, content priority, touch targets, производительность.

## When to Use
- Проектирование и разработка интерфейсов в категории responsive
- Работа над продуктами: mobile, web-app, saas, ecommerce, landing
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Старт с mobile (320-375px), расширять до desktop
4. - Content priority: самое важное первым на mobile
5. - Touch targets: 44x44pt с самого начала
6. - Progressive enhancement: не graceful degradation
