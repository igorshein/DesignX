# Design Strategy & Framing

## Source
DesignX Catalog · ux-research

## Purpose
Problem framing through the Five Foundational Questions. Research synthesis, opportunity sizing, hypothesis definition, competitive analysis.

## When to Use
- Проектирование и разработка интерфейсов в категории ux-research
- Работа над продуктами: web-app, saas, mobile, landing, ecommerce
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
   Frame problem before designing solution
   Validate problem exists before solving
   Define audience by needs, not demographics
   Test solution fit before building features
   Know the competitive landscape
