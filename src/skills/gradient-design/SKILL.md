# SKILL: Gradient Design

## Source
`DesignX Catalog`

## Purpose
Использование градиентов в UI: линейные и радиальные для фонов, conic для спиннеров, mesh для глубины. Избегание пурпурных AI-паттернов.

## When to Use
- Проектирование и разработка интерфейсов в категории color-theory
- Работа над продуктами: landing, marketing, brand, mobile
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Linear gradient для фонов и кнопок
4. - Radial gradient для акцентов и глубины
5. - Conic gradient для палитр и загрузчиков
6. - Mesh gradient: 3+ цвета для органичных фонов
