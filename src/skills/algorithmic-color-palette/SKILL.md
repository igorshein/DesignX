# Algorithmic Color Palette

## Source
DesignX Catalog · color-theory

## Purpose
Derive states and brand-tinted greys algorithmically. 11-shade scale, semantic color derivation, accessible contrast verification.

## When to Use
- Проектирование и разработка интерфейсов в категории color-theory
- Работа над продуктами: web-app, saas, landing, mobile
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
   Start with brand color, derive everything
   Generate 11-shade scale from single hue
   Derive hover/active/disabled states algorithmically
   Tint greys with brand hue for cohesion
   Verify contrast at every step
