# AI Personalization & Ethics

## Source
DesignX Catalog · interaction

## Purpose
Фреймворк ADAPT: Personalization is a power dynamic. Adaptive interfaces, privacy balance, filter bubble prevention, ethical AI design.

## When to Use
- Проектирование и разработка интерфейсов в категории interaction
- Работа над продуктами: web-app, mobile, saas, ecommerce
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
   Give users agency over personalization
   Be transparent about data usage
   Prevent filter bubbles
   Allow opt-out easily
   Test for bias regularly
