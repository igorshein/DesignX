# SKILL: Mobile UI Design

## Source
`designer-skills` · [GitHub](https://github.com/Owl-Listener/designer-skills)

## Purpose
Проектирование мобильных интерфейсов с учетом Apple HIG и Material Design 3: safe areas, touch targets 44pt, жесты, нативная навигация.

## When to Use
- Проектирование и разработка интерфейсов в категории mobile
- Работа над продуктами: mobile
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Центры кнопок iOS: 60pt друг от друга
4. - Android: архитектура, темы, safe data handling
5. - Touch targets: 44pt высота, padding для комфорта
6. - Адаптация под notch, Dynamic Island, camera cutouts
