# SKILL: Design Handoff Communication

## Source
DesignX Catalog · design-ops

## Purpose
Коммуникация передачи дизайна разработке: specs, annotation, design review, developer handoff, collision avoidance.

## When to Use
- Проектирование и разработка интерфейсов в категории design-ops
- Работа над продуктами: web-app, saas, mobile, desktop, dashboard
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Specs: размеры, отступы, состояния, responsive поведение
4. - Annotation: пояснения к решениям, edge cases, fallbacks
5. - Design review: синхронный/асинхронный, checklist, approval
6. - Collision avoidance: token system, component library, single source
