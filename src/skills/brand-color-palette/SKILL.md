# SKILL: Brand Color Palette

## Source
DesignX Catalog · color-theory

## Purpose
Разработка бренд-палитры: primary (основной), secondary (вспомогательный), accent (акцентный), neutral (фоны, текст), semantic (success, error, warning). Шкала 50-900 для каждого.

## When to Use
- Проектирование и разработка интерфейсов в категории color-theory
- Работа над продуктами: web-app, landing, saas, mobile, ecommerce, brand, marketing
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Primary: основной цвет бренда, CTA, заголовки
4. - Secondary: дополнительный, менее заметный
5. - Accent: акцентный, для выделения (1-2 цвета)
6. - Neutral: фоны 50-200, текст 800-900, border 300-400
