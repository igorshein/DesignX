# SKILL: Design Mockup Presentation

## Source
DesignX Catalog · visual-design

## Purpose
Презентация дизайн-макетов: mockup presentation, case study, портфолио, сторителлинг через визуальные решения.

## When to Use
- Проектирование и разработка интерфейсов в категории visual-design
- Работа над продуктами: web-app, mobile, saas, landing, dashboard
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Структура: problem → process → solution → impact
4. - Визуальная подача: clean mockups, context screens, flow diagrams
5. - Case study: обоснование решений, метрики, A/B результаты
6. - Портфолио: narrative arc, personality, проектная фильтрация
