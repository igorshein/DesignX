# SKILL: UX Review

## Source
`designer-skills` · Nielsen Norman Group

## Purpose
Комплексное UX-ревью на основе эвристик NN/g и принципов визуального дизайна.

## When to Use
При редизайне, аудите существующего продукта, или перед крупным релизом.

## Behavior
1. Примени эвристики Nielsen Norman Group (10 эвристик)
2. Проанализируй scale, visual hierarchy, balance, contrast, Gestalt-принципы
3. Оцени пользовательские потоки и информационную архитектуру
4. Проверь консистентность: термины, элементы, паттерны
5. Сформулируй рекомендации с приоритизацией (P0, P1, P2)
6. Каждую рекомендацию подкрепи ссылкой на принцип или исследование
