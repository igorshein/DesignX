# AI Agent UX

## Source
`varunk130/ai-ux-skill-library` · [GitHub](https://github.com/varunk130/ai-ux-skill-library)

## Purpose
Фреймворк AUTONOMY: Users supervise, AI executes. Autonomy controls, consent, action previews, undo/rollback, audit trails, interruptability.

## When to Use
- Проектирование и разработка интерфейсов в категории interaction
- Работа над продуктами: web-app, saas, desktop
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
   Users supervise, AI executes
   Clear action previews before execution
   One-click undo and rollback
   Audit trails for all agent actions
   Progressive autonomy based on trust
