# SKILL: Voice UI Design

## Source
`DesignX`

## Purpose
Голосовые интерфейсы: conversation design, VUI patterns, speech recognition UX, voice feedback, multimodal voice+screen.

## When to Use
- Проектирование и разработка интерфейсов в категории interaction
- Работа над продуктами: mobile, web-app, desktop, game
- Проектирование голосовых сценариев

## Behavior
1. Осмысли контекст: платформу, аудиторию, голосовые сценарии, окружение
2. Следуй ключевым принципам:
3. - Дизайн для ушей, а не для глаз
4. - Подтверждай критические действия голосом
5. - Обрабатывай тишину и неоднозначность
6. - Визуальная обратная связь для голосового ввода
7. - Предлагай альтернативы при ошибке распознавания
