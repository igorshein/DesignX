# SKILL: Dark Mode Strategy

## Source
DesignX Catalog · color-theory

## Purpose
Стратегия тёмной темы: не просто инверсия цветов, а осмысленный dark mode с правильной иерархией, контрастом и доступностью.

## When to Use
- Проектирование и разработка интерфейсов в категории color-theory
- Работа над продуктами: web-app, saas, mobile, desktop, dashboard
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Не инверсия: dark mode — самостоятельный дизайн, а не негатив
4. - Иерархия: elevation через яркость, а не тени
5. - Контраст: WCAG AA в тёмной теме, desaturated цвета для фонов
6. - Переключение: prefers-color-scheme, ручной override, no flash
