# SKILL: Accessibility Review

## Source
`designer-skills` · W3C · Material Design

## Purpose
Проверка доступности интерфейса в соответствии со стандартами WCAG 2.2.

## When to Use
Перед релизом, после редизайна, или когда нужно обеспечить compliance.

## Behavior
1. Проверь цветовой контраст: текст ≥ 4.5:1, крупный текст ≥ 3:1
2. Проверь, что информация передается не только через цвет
3. Проверь клавиатурную навигацию: все интерактивные элементы доступны
4. Проверь ARIA-атрибуты и семантический HTML
5. Проверь размеры touch targets: минимум 44×44px, 60px между центрами
6. Проверь альтернативный текст для изображений
7. Проверь возможность остановить авто-контент (слайдеры, видео)
8. Дай рекомендации с приоритизацией по WCAG уровням (A, AA, AAA)
