# SKILL: Competitive Analysis

## Source
`DesignX Catalog`

## Purpose
Конкурентный анализ: прямые и косвенные конкуренты, UX-паттерны, features, SWОT, feature matrix, gap analysis для дифференциации.

## When to Use
- Проектирование и разработка интерфейсов в категории ux-research
- Работа над продуктами: web-app, saas, mobile, ecommerce, landing
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Прямые (тот же use case) + косвенные (альтернативы)
4. - Оси: UX, features, визуал, цена, аудитория
5. - SWOT: strengths, weaknesses, opportunities, threats
6. - Feature matrix: у кого что есть
