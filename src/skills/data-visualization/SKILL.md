# SKILL: Data Visualization Design

## Source
DesignX Catalog · data-viz

## Purpose
Проектирование data-heavy интерфейсов: дашборды, KPI-панели, графики, таблицы. Выбор типа визуализации, иерархия, адаптивность.

## When to Use
- Проектирование и разработка интерфейсов в категории data-viz
- Работа над продуктами: dashboard, web-app, saas
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Тип визуализации под задачу
4. - Иерархия: важное наверху
5. - Tabular figures для цифр
6. - Не полагаться только на цвет
