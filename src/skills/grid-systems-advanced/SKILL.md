# SKILL: Grid Systems Advanced

## Source
DesignX Catalog · visual-design

## Purpose
Продвинутые техники построения сеток: модульные сетки, golden ratio в композиции, baseline grid для типографики, custom grid generation, responsive grid адаптация, subgrid, grid systems для сложных дашбордов и editorial layouts.

## When to Use
- Проектирование и разработка интерфейсов в категории visual-design
- Работа над продуктами: web-app, dashboard, landing, saas, editorial
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Выбирай тип сетки под контент, а не наоборот
4. - Golden ratio: 1:1.618 для гармоничных пропорций
5. - Baseline grid: интерлиньяж кратен базовой линии
6. - Модульная сетка: ячейки, колонки, gutter — единая система
7. - Адаптируй сетку: 4-колонки mobile → 8-12 колонок desktop
