# SKILL: Adaptive Design

## Source
DesignX Catalog · responsive

## Purpose
Adaptive vs responsive: фиксированные макеты для каждого breakpoint. Feature detection, device-specific UX, server-side adaptation (RESS).

## When to Use
- Проектирование и разработка интерфейсов в категории responsive
- Работа над продуктами: web-app, mobile, ecommerce, desktop
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Фиксированные макеты для каждого breakpoint
4. - Feature detection: CSS supports, @supports
5. - Device-specific UX: камера, GPS, touch на mobile
6. - RESS: server-side адаптация контента

adaptive, device-specific, breakpoints, layouts
