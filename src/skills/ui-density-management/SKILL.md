# UI Density Management

## Source
DesignX Catalog · layout-composition

## Purpose
Match UI density to platform and user type. Compact vs comfortable spacing, data density options, user-controlled density, platform conventions.

## When to Use
- Проектирование и разработка интерфейсов в категории layout-composition
- Работа над продуктами: web-app, desktop, saas, dashboard
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
   Experts prefer dense, novices need space
   Match density to platform (desktop denser)
   Offer density controls for complex tools
   Data displays benefit from compact mode
   Touch targets need 44x44px regardless of density
