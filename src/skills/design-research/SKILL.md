# SKILL: Design Research

## Source
`designer-skills` · [GitHub](https://github.com/Owl-Listener/designer-skills)

## Purpose
Проведение и анализ пользовательских исследований: планирование интервью, usability-тестирование, анализ данных, journey maps, формулирование инсайтов.

## When to Use
- Проектирование и разработка интерфейсов в категории ux-research
- Работа над продуктами: web-app, saas, mobile, ecommerce, desktop
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Соединение команды с реальными пользователями
4. - Методы: интервью, дневники, контекстный инквайри
5. - Инструменты: Calendly, Typeform, UserTesting, Dovetail
6. - Построение journey maps и user flows
