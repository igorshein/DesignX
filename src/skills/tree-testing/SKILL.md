# SKILL: Tree Testing

## Source
`DesignX Catalog`

## Purpose
Проверка IA: пользователи ищут контент в текстовом дереве. Метрики: success rate (>80%), directness, time. Итеративное улучшение до дизайна.

## When to Use
- Проектирование и разработка интерфейсов в категории ux-research
- Работа над продуктами: web-app, saas, mobile, ecommerce
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Текстовое дерево, без визуала
4. - Конкретные задачи, не «найдите»
5. - Метрики: success rate, directness, time
6. - 10-20 участников на тест
