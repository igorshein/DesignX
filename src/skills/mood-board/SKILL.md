# SKILL: Mood Board Creation

## Source
DesignX Catalog · visual-design

## Purpose
Коллажи из референсов, палитр, шрифтов и текстур для согласования визуального направления с командой и стейкхолдерами.

## When to Use
- Проектирование и разработка интерфейсов в категории visual-design
- Работа над продуктами: landing, marketing, brand, saas, game
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Референсы: сайты, фото, принты, текстуры
4. - 5-7 основных цветов из референсов
5. - 2-3 шрифта под настроение
6. - 5-10 ключевых слов эстетики
