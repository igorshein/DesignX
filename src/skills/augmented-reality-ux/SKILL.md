# SKILL: AR UX Design

## Source
`DesignX`

## Purpose
Дизайн дополненной реальности: spatial interfaces, AR interactions, world-locked UI, environment understanding.

## When to Use
- Проектирование и разработка интерфейсов в категории prototyping
- Работа над продуктами: mobile, web-app, game
- Проектирование AR-сценариев

## Behavior
1. Осмысли контекст: тип AR, окружение, устройство, взаимодействие
2. Следуй ключевым принципам:
3. - Цифровой контент уважает физическое пространство
4. - World-locked для персистентности, screen-locked для контролов
5. - Occlusion закрепляет AR в реальности
6. - Направляй пользователя сканировать окружение
7. - Дизайн для комфорта: избегай быстрых движений
