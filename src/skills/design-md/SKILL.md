# SKILL: DESIGN.md Creator

## Source
`google-labs-code` · [GitHub](https://github.com/VoltAgent/awesome-agent-skills)

## Purpose
Создание и управление файлами DESIGN.md — текстовой системы дизайна для ИИ-агентов.

## When to Use
Когда проект не имеет описанной системы дизайна, или нужно конвертировать существующий бренд-гайд в agent-friendly формат.

## Behavior
1. Собери бренд-информацию: цвета, шрифты, отступы, иконки
2. Структурируй в секции: Brand & Tone, Visual Aesthetic, Color Tokens, Typography, Spacing, Components, Motion, Accessibility
3. Используй Markdown-таблицы для токенов
4. Опиши семантику: какой токен для чего (surface, text primary, accent)
5. Укажи анти-паттерны и что НЕЛЬЗЯ делать
6. Добавь ссылки на источники (Figma, бренд-гайд)
