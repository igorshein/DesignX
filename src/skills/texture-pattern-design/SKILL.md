# SKILL: Texture & Pattern Design

## Source
DesignX Catalog · visual-design

## Purpose
Дизайн и применение текстур и паттернов в интерфейсах: CSS-паттерны, SVG-паттерны, noise texture, paper texture, fabric texture, geometric patterns, organic patterns, seamless tile generation, performance-optimised текстуры для веба.

## When to Use
- Проектирование и разработка интерфейсов в категории visual-design
- Работа над продуктами: landing, marketing, brand, web-app
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Текстуры дают глубину без дополнительных элементов
4. - Seamless tiles: бесшовное повторение без видимых стыков
5. - Noise texture: микро-текстура для борьбы с flat-стерильностью
6. - CSS/SVG паттерны легче изображений — используй их
7. - Не перегружай: текстура — фон, не контент
