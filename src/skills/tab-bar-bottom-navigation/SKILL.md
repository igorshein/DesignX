# SKILL: Tab Bar & Bottom Navigation

## Source
`DesignX Catalog`

## Purpose
Дизайн мобильной навигации: tab bar, bottom navigation, iOS/Android conventions, active states, badges.

## When to Use
- Проектирование и разработка интерфейсов в категории interaction
- Работа над продуктами: mobile, web-app, saas, dashboard
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Tab bar: максимум 5 вкладок, always-on, активное состояние
4. - Bottom navigation: safe area, reachability, thumb zone
5. - iOS vs Android: HIG guidelines vs Material Design conventions
6. - Badges: уведомления, индикаторы, counter, dot
