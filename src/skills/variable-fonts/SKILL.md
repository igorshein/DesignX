# SKILL: Variable Fonts

## Source
`DesignX`

## Purpose
Кастомизация вариативных шрифтов: font axes (wght, wdth, opsz, slnt, ital), оптический размер, производительность, анимация осей.

## When to Use
- Проектирование и разработка интерфейсов в категории typography
- Работа над продуктами: web-app, landing, saas, mobile, marketing
- Выбор и настройка типографики

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, требования к производительности
2. Следуй ключевым принципам:
3. - Один вариативный шрифт заменяет множество статических начертаний
4. - Optical sizing адаптируется под размер отображения
5. - Анимация осей для микро-взаимодействий
6. - Subset шрифтов для производительности
7. - Тестируй читаемость на крайних значениях осей
