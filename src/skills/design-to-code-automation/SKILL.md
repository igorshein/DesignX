# SKILL: Design-to-Code Automation

## Source
DesignX Catalog · automation

## Purpose
Автоматическая генерация кода из дизайна: design token pipelines, codegen, handoff automation. Устраняет ручной перенос дизайна в код.

## When to Use
- Проектирование и разработка интерфейсов в категории frontend-dev
- Работа над продуктами: web-app, saas, mobile, desktop
- Настройка пайплайнов дизайн → код

## Behavior
1. Осмысли контекст: дизайн-систему, токены, целевой фреймворк
2. Следуй ключевым принципам:
3. - Design tokens — единый источник истины
4. - Генерируй код, а не скриншоты
5. - Responsive constraints → media queries
6. - Состояния (hover/active/disabled) генерируются автоматически
7. - Пайплайн запускается при сохранении дизайна
