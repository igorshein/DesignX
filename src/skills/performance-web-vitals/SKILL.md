# Performance & Web Vitals

## Source
`dembrandt/dembrandt-skills` · [GitHub](https://github.com/dembrandt/dembrandt-skills)

## Purpose
Lighthouse audit, LCP under 2.5s, CLS under 0.1, INP under 200ms, image optimization, font loading, JS bundle optimization, perceived performance.

## When to Use
- Проектирование и разработка интерфейсов в категории frontend-dev
- Работа над продуктами: web-app, landing, ecommerce, saas
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
   LCP under 2.5s for good UX
   CLS under 0.1 for visual stability
   INP under 200ms for responsiveness
   Optimize images (WebP, responsive)
   Lazy-load below-fold content
