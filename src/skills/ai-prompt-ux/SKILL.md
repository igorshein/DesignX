# AI Prompt UX

## Source
DesignX Catalog · interaction

## Purpose
Фреймворк CRAFT: A blank text box is not prompt UX. Input affordances, templates, suggestions, history, progressive prompting.

## When to Use
- Проектирование и разработка интерфейсов в категории interaction
- Работа над продуктами: web-app, mobile, saas
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
   Constrain input to guide quality output
   Reveal capabilities through suggestions
   Assist with templates and examples
   Format output for consumption
   Teach users how to prompt effectively
