# SKILL: Micro-interactions Design

## Source
`DesignX Catalog`

## Purpose
Структура: trigger → rules → feedback → loops. 100-300ms, ease-out/spring, haptic feedback. Примеры: like, toggle, pull to refresh.

## When to Use
- Проектирование и разработка интерфейсов в категории interaction
- Работа над продуктами: web-app, mobile, saas, ecommerce, game
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Структура: trigger → rules → feedback → loops
4. - Duration: 100-300ms для feedback
5. - Easing: ease-out, spring для natural feeling
6. - Feedback: визуальный + звуковой + тактильный
