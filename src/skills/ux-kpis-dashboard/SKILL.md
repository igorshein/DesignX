# SKILL: UX KPIs Dashboard

## Source
`DesignX`

## Purpose
Метрики успеха UX: NPS, SUS, CES, task success rate. Проектирование KPI-дашбордов для продуктовых команд.

## When to Use
- Проектирование и разработка интерфейсов в категории ux-research
- Работа над продуктами: web-app, saas, dashboard, mobile, ecommerce
- Построение системы метрик продукта

## Behavior
1. Осмысли контекст: тип продукта, бизнес-цели, пользовательские сценарии
2. Следуй ключевым принципам:
3. - Выбирай метрики, привязанные к пользовательским целям
4. - NPS измеряет лояльность, SUS — юзабилити, CES — усилия
5. - Триангулируй количественные и качественные данные
6. - Дашборд рассказывает историю, а не просто числа
7. - Устанавливай целевые значения и отслеживай тренды
