# SKILL: Usability Benchmarking

## Source
`DesignX`

## Purpose
Бенчмаркинг юзабилити: comparative testing, SUS benchmarking, competitive UX analysis, baseline metrics.

## When to Use
- Проектирование и разработка интерфейсов в категории ux-research
- Работа над продуктами: web-app, saas, mobile, ecommerce, dashboard
- Сравнительный анализ пользовательского опыта

## Behavior
1. Осмысли контекст: конкурентов, стандартные задачи, метрики
2. Следуй ключевым принципам:
3. - Сравнивай с прямыми конкурентами и best-in-class
4. - Стандартизируй задачи для валидного сравнения
5. - SUS score выше 68 — выше среднего
6. - Отслеживай метрики в динамике
7. - Комбинируй количественные и качественные данные
