# SKILL: Menu Navigation Advanced

## Source
`DesignX Catalog`

## Purpose
Проектирование сложной навигации: mega-menu, breadcrumbs, вертикальная/горизонтальная иерархия, многоуровневые меню.

## When to Use
- Проектирование и разработка интерфейсов в категории interaction
- Работа над продуктами: web-app, saas, dashboard, ecommerce, desktop
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - Mega-menu: многоуровневая группировка, категории, визуальные подсказки
4. - Breadcrumbs: иерархическая и историческая навигация
5. - Вертикальная навигация: sidebar, nested accordion, collapsible
6. - Горизонтальная навигация: tabs, top bar, overflow menu

## Tags
navigation, mega-menu, breadcrumbs, menu, hierarchy

## For Product Types
