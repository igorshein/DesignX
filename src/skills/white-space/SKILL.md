# SKILL: White Space Management

## Source
`DesignX Catalog`

## Purpose
Осознанное использование пустого пространства: микро (gap 8-24px), макро (секции 48-96px), внутренний padding карточек (16-24px), 8px base unit.

## When to Use
- Проектирование и разработка интерфейсов в категории layout-composition
- Работа над продуктами: web-app, dashboard, landing, saas, mobile, ecommerce, desktop
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - 8px base unit: все отступы кратны 8
4. - Макро-пространство: 48-96px между секциями
5. - Микро-пространство: 8-24px между элементами
6. - Пэддинг карточек: 16-24px

## Tags
whitespace, spacing, padding, margin, breathing
