# SKILL: Push Notification UX

## Source
DesignX Catalog · notifications

## Purpose
Стратегия push-уведомлений: timing, персонализация, permission requests, opt-in оптимизация, frequency capping.

## When to Use
- Проектирование и разработка интерфейсов в категории interaction
- Работа над продуктами: mobile, web-app, saas, ecommerce
- Проектирование системы уведомлений

## Behavior
1. Осмысли контекст: платформу, типы уведомлений, поведение пользователя
2. Следуй ключевым принципам:
3. - Запрашивай разрешение в контексте, а не при запуске
4. - Персонализируй контент на основе поведения
5. - Уважай frequency capping и тихие часы
6. - Rich-уведомления с action-кнопками
7. - Тестируй время отправки для максимального вовлечения
