# SKILL: Flexbox Layout

## Source
`DesignX Catalog`

## Purpose
Flexbox: flex-direction, justify-content, align-items, flex-wrap, gap, flex-grow/shrink/basis. Для компонентов, навигации, карточек, центрирования.

## When to Use
- Проектирование и разработка интерфейсов в категории frontend-dev
- Работа над продуктами: web-app, dashboard, landing, saas, mobile, ecommerce, desktop
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - flex-direction: row (дефолт) или column
4. - justify-content: main axis распределение
5. - align-items: cross axis выравнивание
6. - flex-wrap: перенос на новую строку

## Tags
flexbox, flex, alignment, distribution, layout

## For Product Types
web-app, dashboard, landing, saas, mobile, ecommerce, desktop
