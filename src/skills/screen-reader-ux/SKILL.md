# SKILL: Screen Reader UX

## Source
DesignX Catalog · accessibility

## Purpose
Тестирование и дизайн для screen reader: NVDA, VoiceOver, JAWS, ARIA-атрибуты, reading order, keyboard navigation.

## When to Use
- Проектирование и разработка интерфейсов в категории accessibility
- Работа над продуктами: web-app, saas, mobile, desktop, dashboard
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
3. - NVDA: Windows, Firefox/Chrome, free, open-source
4. - VoiceOver: macOS/iOS, встроенный, gesture-based навигация
5. - JAWS: Windows, платный, enterprise-стандарт
6. - ARIA: landmarks, live regions, alert roles, accessible names
