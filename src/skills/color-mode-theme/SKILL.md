# Color Mode & Theme Design

## Source
DesignX Catalog · color-theory

## Purpose
Light vs dark vs combined mode. Theme selector strategy, color scheme adaptation, semantic token swapping, prefers-color-scheme detection.

## When to Use
- Проектирование и разработка интерфейсов в категории color-theory
- Работа над продуктами: web-app, saas, mobile, desktop
- Интеграция с дизайн-системами и ИИ-агентами

## Behavior
1. Осмысли контекст: тип продукта, аудиторию, платформу, технические ограничения
2. Следуй ключевым принципам:
   Design light first, derive dark
   Use semantic tokens, swap per mode
   Respect prefers-color-scheme
   Allow manual override
   Test both modes for contrast
