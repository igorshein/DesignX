DesignX v2

The largest open catalog of UI/UX design skills and DESIGN.md design systems for AI agents.

Difference from Open Design: DesignX is a recommendation catalog, not an artifact generator. We help you understand which skills and design systems your project needs.

How to use

  1. Select product type → DesignX picks skills + design system
  2. Select design philosophy → refine the style
  3. Export set → SKILL.md + DESIGN.md to repo root
  4. AI agent reads files and follows the design system
